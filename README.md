# **Basic .NET Console Application**

This is a small .NET console program that shows how to take input from the user, show output, and do a basic calculation.

---

## **What You Need**

Before running the program, make sure you have:

* **.NET SDK 8.0 or newer**
  (Download from the Microsoft website)

---

## **How to Run the Program**

1. **Restore the project** (if required):

   ```
   dotnet restore
   ```

2. **Build the project**:

   ```
   dotnet build
   ```

3. **Run the project**:

   ```
   dotnet run
   ```

   You can also directly run without building:

   ```
   dotnet run
   ```

---

## **What This Program Does**

* Greets the user
* Asks for your name
* Says hello using your name
* Takes a number from you
* Doubles the number and shows the result

---

## **Project Files**

* **Program.cs** → The main C# file where all the code runs
* **BasicConsoleApp.csproj** → The project file that defines the .NET setup
