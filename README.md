# Basic .NET Console Application

A simple, working .NET console application that demonstrates basic input/output and calculations.

## Prerequisites

- .NET SDK 8.0 or later installed on your system
- You can download it from: https://dotnet.microsoft.com/download

## How to Run

1. **Restore dependencies** (if needed):
   ```
   dotnet restore
   ```

2. **Build the project**:
   ```
   dotnet build
   ```

3. **Run the application**:
   ```
   dotnet run
   ```

Or you can combine build and run:
```
dotnet run
```

## What This Application Does

- Welcomes the user
- Asks for your name and greets you
- Performs a simple calculation (doubles a number you provide)

## Project Structure

- `Program.cs` - Main entry point of the application
- `BasicConsoleApp.csproj` - Project file defining the .NET project
